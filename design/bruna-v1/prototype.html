flowtype vs Wispr Flow

O mesmo padrão de UX, sem o cap de tokens.

Critério Wispr Flow flowtype
PreçoPago após N tokensFree, sem limite
STT engineCloud onlyGroq pool (3 keys) + local fallback
Privacy fallbackÁudio sempre saiModo local-only disponível
Vocab customLimitadoLista global ou per-app
Histórico searchableSim, sem busca FTSFTS5 + export md/json
Open sourceNãoSim (MIT)

Features

Tudo que o Wispr tem, mais o que faltava.

⌨

Hotkey global

Right Ctrl default, customizável. Hold detection via uIOhook.

⚡

Sub-1s STT

Groq Whisper Large v3 Turbo. p50 720ms hotkey→paste.

☁☁☁

Pool de 3 keys

Round-robin entre 3 Groq keys = 43.2k req/dia grátis.

💻

Fallback local

faster-whisper small.en bundled. Funciona offline.

📚

Vocabulário custom

Correções term_wrong→term_correct, global ou per-app.

🕘

Histórico FTS5

Busca full-text, filtros por data/app, export md/json.

🎯

Overlay 4 estados

idle, armed, capturing, processing. Always-on-top, slim.

⏻

Auto-start Windows

Liga junto com o Windows. Roda silencioso no tray.
